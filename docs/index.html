Three Layers of Protection

Noirva uses a multi-layered approach to ensure you never see an ad again.

Network Rules

Blocks ad requests before they even load. Our curated rule list stops tracking scripts, analytics, and ad networks at the network level.

API Filtering

Intercepts YouTube's API responses and strips out ad placements, sponsored content, and promotional data before it reaches your screen.

DOM Cleanup

Removes any remaining ad elements from the page. Even if something slips through, our DOM layer cleans up the visual clutter.

How It Works

1

Network Interception

When you load YouTube, Noirva's network rules immediately block connections to ad servers, tracking pixels, and analytics endpoints.

2

Response Filtering

As YouTube's API responses arrive, Noirva parses them in real-time and removes any ad placements, sponsored cards, or promotional content.

3

DOM Cleanup

Finally, Noirva scans the rendered page and removes any remaining ad elements, placeholders, or promotional banners that made it through.

Frequently Asked Questions

Is Noirva safe to use?

Yes. Noirva is open-source and only blocks ad-related network requests. It doesn't collect any data or track your browsing.

Will this break YouTube?

No. Noirva only blocks ads. Videos, comments, likes, and all other YouTube features work normally.

How do I install the Android APK?

Download the APK, then enable "Install from Unknown Sources" in your device's Settings → Security. Open the APK file to install.

macOS says the app is damaged. What do I do?

Right-click (or Control-click) the Noirva app and select "Open". This bypasses Gatekeeper on the first launch. The app is signed and safe.
