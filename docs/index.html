// HOW IT WORKS

From tap to ad-free

1

Network interception

When you load YouTube, Noirva's network rules immediately block connections to ad servers, tracking pixels, and analytics endpoints.

2

Response filtering

As YouTube's API responses arrive, Noirva parses them in real time and removes ad placements, sponsored cards, and promotional content.

3

DOM cleanup

Finally, Noirva scans the rendered page and removes any remaining ad elements, placeholders, or banners that made it through.
